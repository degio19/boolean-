TIPI DI TAG

i tag si dividono in 3 tipi:

Block Level tag

il contenuto prende tutta la linea, la larghezza della pagina. es< p>qualcosa< p>

In line Tag

prende solo lo spazio che richiede il contenuto es. < strong> qualcosa < strong>

Self Closing Tag

sono quei comandi che non richiedono chiusura es.: < br>

LISTA ORDINATA

si fa usando il comando < ol> < /ol> ordinded list o < ul> < /ul> unordered list, ogni elemento della lista si chiama poi con il tag < li>

esempio lista ordinata

  1. cane
  2. gatto

esempio lista non ordinata

  • cane
  • gatto
